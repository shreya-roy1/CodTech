# Full-Stack Development Task Series

A collection of four technical deliverables ranging from front-end UI/UX design to real-time asynchronous communication systems.

## 🚀 Projects Overview

| Task | Project | Tech Stack | Key Features |
| :--- | :--- | :--- | :--- |
| **01** | **Interactive Quiz App** | JavaScript (ES6+), CSS3 | Dynamic scoring, instant feedback, DOM manipulation. |
| **02** | **Real-Time Chat App** | React.js, Socket.io, Node.js | Bi-directional communication, message history, responsive UI. |
| **03** | **Personal Portfolio** | HTML5, CSS3 (Grid/Flexbox) | Single-page layout, smooth scroll, mobile-responsive. |
| **04** | **E-Learning Platform** | React, Tailwind CSS | Video embedding, progress tracking, glassmorphism UI. |

---

## 🛠 Project Deep Dives

### Task 1: Interactive Quiz Application
A lightweight, logic-driven application that handles state management for user scores and dynamic question rendering.
*   **Highlight:** Uses a centralized data structure for questions, allowing for easy content expansion without changing the logic.

### Task 2: Real-Time Chat Engine
A full-stack implementation of the WebSocket protocol to facilitate instant messaging.
*   **Architecture:** Node.js backend broadcasts events to the React frontend in under 50ms.
*   **UI:** Includes auto-scroll functionality and time-stamped message bubbles.

### Task 3: Personal Developer Portfolio
A polished, professional landing page designed to convert visitors into collaborators.
*   **Design Philosophy:** Minimalist dark theme with high-contrast accent colors.
*   **Responsiveness:** Fully optimized for Mobile, Tablet, and Desktop using CSS Media Queries.

### Task 4: E-Learning Platform UI
A high-fidelity interface focused on the student experience.
*   **Visual Style:** Features a modern **Glassmorphism** aesthetic with solid-fill progress indicators.
*   **Components:** Includes a sidebar curriculum navigator and a distraction-free video player environment.
